# Password Strength Checker (Web Version)

This is a simple web-based password strength checker built using HTML, CSS, and JavaScript. It analyzes the entered password and classifies it as Weak, Medium, or Strong based on character types and length.

## Features
- Checks uppercase, lowercase, digits, and special characters
- Evaluates password length
- Displays strength in real-time on button click
- Fully responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the project
2. Open the folder
3. Run the file:
   index.html
4. The project will open in your browser
